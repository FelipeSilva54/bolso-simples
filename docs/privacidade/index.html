1. Introdução e identificação do responsável

O Bolso Simples é um aplicativo de controle financeiro pessoal desenvolvido e mantido por Felipe Silva (doravante "nós", "nosso" ou "desenvolvedor").

Esta Política de Privacidade descreve como coletamos, utilizamos, armazenamos e protegemos as informações pessoais dos usuários do aplicativo Bolso Simples disponível para Android (pacote: com.bolsosimples.app).

Ao utilizar o aplicativo, você concorda com as práticas descritas neste documento. Caso não concorde com qualquer parte desta política, recomendamos que não utilize o aplicativo.

Nosso compromisso: O Bolso Simples é 100% gratuito, sem anúncios e sem rastreamento de comportamento. Seus dados financeiros pertencem exclusivamente a você.

2. Dados que coletamos

Coletamos apenas os dados estritamente necessários para o funcionamento do aplicativo:

2.1 Dados de autenticação — Login com Google

Quando você opta por entrar com sua conta Google, recebemos do Google as seguintes informações:

  • Nome completo — exibido no perfil do aplicativo
  • Endereço de e-mail — utilizado como identificador de conta
  • Foto de perfil (URL) — exibida no perfil do aplicativo
  • ID único (UID) gerado pelo Firebase — identifica sua conta de forma anônima nos servidores

Não armazenamos senhas. A autenticação é realizada integralmente pelo serviço Firebase Authentication (Google), por meio de um token de acesso seguro (OAuth 2.0).

2.2 Dados de autenticação — Login anônimo

Caso escolha entrar sem uma conta, o Firebase Authentication gera automaticamente um identificador único anônimo. Nenhuma informação pessoal identificável é coletada neste modo. O acesso anônimo permite o uso completo do aplicativo, mas os dados não podem ser recuperados caso o aplicativo seja desinstalado.

2.3 Dados financeiros inseridos por você

Todos os dados financeiros são criados e inseridos manualmente por você. Coletamos:

Tipo Dados Onde é armazenado
Carteiras Nome, cor escolhida, saldo, data de criação Firebase Firestore (nuvem)
Transações Tipo (receita/despesa/transferência), título, descrição, valor, categoria, status, data, informações de parcelamento e recorrência Firebase Firestore (nuvem)
Categorias Nome, ícone, cor, tipo (receita/despesa) Firebase Firestore (nuvem)

2.4 Preferências do dispositivo

Armazenamos localmente no seu dispositivo (sem envio à nuvem) as seguintes preferências:

  • Idioma — Português ou Inglês
  • Moeda padrão — BRL, USD, EUR, entre outras
  • Tema visual — claro ou escuro
  • Visibilidade do saldo — mostrar ou ocultar

2.5 Notificações internas

O aplicativo gera notificações informativas localmente no seu dispositivo (ex.: despesas com vencimento próximo, saldo negativo). Essas notificações são armazenadas apenas no próprio dispositivo e não são enviadas a servidores externos.

O que NÃO coletamos: Localização geográfica · Câmera ou fotos · Contatos · Microfone · Dados bancários ou números de cartão · Informações demográficas · Comportamento de navegação · Dados de telemetria ou crash reports.

3. Como usamos seus dados

Utilizamos seus dados exclusivamente para:

  • Autenticar sua identidade e garantir que apenas você acesse suas informações
  • Armazenar e sincronizar suas carteiras, transações e categorias entre dispositivos
  • Exibir seu nome e foto no perfil do aplicativo (apenas usuários Google)
  • Calcular saldos e resumos financeiros exibidos no aplicativo
  • Personalizar a experiência com base nas suas preferências de idioma, moeda e tema

Não utilizamos seus dados para: publicidade, análise de comportamento, treinamento de modelos de inteligência artificial, venda a terceiros ou qualquer finalidade além das listadas acima.

4. Armazenamento e segurança

4.1 Infraestrutura em nuvem (Firebase)

Os dados de autenticação e financeiros são armazenados nos serviços Firebase Authentication e Firebase Firestore, operados pelo Google. O servidor utilizado está localizado na região southamerica-east1 (São Paulo, Brasil), garantindo que seus dados permaneçam em território nacional.

O Firebase aplica automaticamente criptografia em trânsito (TLS) e em repouso (AES-256). O acesso aos dados é controlado por regras de segurança do Firestore, garantindo que cada usuário acesse somente seus próprios dados.

4.2 Armazenamento local no dispositivo

Preferências e notificações internas são armazenadas localmente usando AsyncStorage, uma biblioteca padrão do React Native. Esses dados ficam no armazenamento interno do seu dispositivo e não são transmitidos a servidores externos.

4.3 Sessão de autenticação

Sua sessão de login é mantida localmente pelo Firebase Authentication com AsyncStorage, evitando que você precise fazer login toda vez que abrir o aplicativo.

Importante: Apesar dos esforços para garantir a segurança dos dados, nenhum sistema é 100% inviolável. Recomendamos que você mantenha seu dispositivo protegido com senha ou biometria.

5. Serviços de terceiros

O Bolso Simples utiliza apenas os seguintes serviços de terceiros:

Google / Firebase

  • Firebase Authentication — gerenciamento de login e sessão
  • Firebase Firestore — banco de dados na nuvem para armazenar seus dados financeiros
  • Google Sign-In SDK — autenticação via conta Google

Ao utilizar o login com Google, você também está sujeito à Política de Privacidade do Google. O Firebase é operado pelo Google e está em conformidade com o GDPR, LGPD e outras regulamentações internacionais de privacidade.

Sem rastreamento: O Bolso Simples não integra nenhum serviço de analytics, publicidade, rastreamento de comportamento, crash reporting (Sentry, Crashlytics) ou qualquer outro serviço de coleta de dados além do Firebase listado acima.

6. Seus direitos (LGPD)

Em conformidade com a Lei Geral de Proteção de Dados (LGPD — Lei nº 13.709/2018), você tem os seguintes direitos em relação aos seus dados pessoais:

6.1 Acesso

Você pode visualizar todos os seus dados financeiros diretamente no aplicativo, a qualquer momento.

6.2 Correção

Você pode editar ou corrigir qualquer informação diretamente no aplicativo (carteiras, transações, categorias).

6.3 Exclusão

Você pode excluir dados específicos (transações, carteiras, categorias) ou excluir sua conta inteiramente através da tela de perfil do aplicativo. A exclusão de conta remove permanentemente todos os seus dados de nossos servidores.

6.4 Portabilidade

Caso deseje exportar seus dados, entre em contato conosco pelo e-mail bolsosimples@gmail.com e processaremos sua solicitação.

6.5 Revogação do consentimento

Você pode revogar o consentimento para tratamento de dados a qualquer momento, excluindo sua conta diretamente no aplicativo ou solicitando via e-mail.

7. Retenção e exclusão de dados

Seus dados são mantidos pelo tempo que você utilizar o aplicativo. Ao excluir sua conta pelo aplicativo, os seguintes dados são removidos permanentemente e de forma imediata:

  • Todas as carteiras e suas transações
  • Todas as categorias personalizadas
  • Dados do perfil (nome, e-mail, foto — referência no Firestore)
  • Conta de autenticação no Firebase
  • Notificações e preferências armazenadas localmente

Dados de sessão gerenciados pelo Firebase Auth podem levar até 90 dias para serem completamente removidos dos sistemas do Google, conforme a política de retenção deles.

8. Menores de idade

O Bolso Simples é um aplicativo destinado a usuários com 13 anos de idade ou mais. Não coletamos intencionalmente dados de crianças menores de 13 anos.

Se você acredita que uma criança menor de 13 anos utilizou o aplicativo e forneceu dados pessoais, entre em contato conosco pelo e-mail bolsosimples@gmail.com para que possamos tomar as medidas necessárias.

9. Alterações nesta política

Podemos atualizar esta Política de Privacidade periodicamente para refletir mudanças no aplicativo ou em requisitos legais. Quando houver alterações relevantes, a data de "Última atualização" no topo desta página será atualizada.

Recomendamos que você revise esta política periodicamente. O uso continuado do aplicativo após alterações constitui aceitação da política atualizada.
